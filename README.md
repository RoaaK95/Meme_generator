# Meme_Generator  
## Summary  
- This webApp is a meme generator, a project created for the Scrimba React course.  
- It takes top and bottom texts then generate a meme by making an API call to get a random image.

## Technologies Used  
- HTML
- CSS
- JavaScript
- React
